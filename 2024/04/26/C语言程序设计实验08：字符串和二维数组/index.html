六、实验心得

字符串操作熟练
掌握了 gets、strlen、字符数组的操作以及字符判断条件。

动态内存管理理解
通过 malloc 与 realloc 动态申请二维数组和字符串的空间，学会处理不同长度数据。

二维数组操作能力增强
熟练输入、遍历二维数组，计算矩阵主副对角线元素和。

字符过滤与数据转换
通过遍历字符数组，将符合条件的数字提取并转换为整数，掌握字符与整数的转换方法。

程序调试能力提升
改错题帮助理解逻辑错误及边界条件处理，提高调试能力。
